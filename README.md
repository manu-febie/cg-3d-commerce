3D Commerce
===========

Requirements
------------
- npm

Dependecies
-----------
- babylonjs

Development dependencies
------------------------
- @babel/core
- @babel/preset-env
- babelify
- browserify
- eslint
- tinify

Local installation
------------------

Clone the repository.

```
git clone https://github.com/manu-febie/cg-3d-commerce
```

Install packages.

```
npm install
```
